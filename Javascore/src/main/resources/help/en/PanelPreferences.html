Java Score - Preferences

In this panel you can do overall program settings. Find it in the 'File' menu.

Misc1 Label: Optional field label in the entries panel. After the first installation you'll find "CBYRA#" or "Rsa" or something similar. This is only a default, which is shipped with Javascore. It's recommended to set this to Membership number for a regional sailing association, but you're free to set any other label.

Misc2 Label: Another optional field label in the entries panel. After the first installation you'll find "US Sailing#" oder "Mna" or something similar. This is only a default, which is shipped with Javascore. It's recommended to set this to Membership number for a national sailing association, but you're free to set any other label

A Factor: Sets an overall custom "A Factor" for use in PHRF Time-on-Time scoring. When either A or B Factor are customized here, they will be used by default when a race using PHRF Time-on-Time is scored. Set the factor to blank or -1 to have it ignored.

B Factor: Sets an overall custom "B Factor" for use in PHRF Time-on-Time scoring. When either A or B Factor are customized here, they will be used by default when a race using PHRF Time-on-Time is scored. Set the factor to blank or -1 to have it ignored.

The factors can also be adjusted in the Race panel under "Advanced Options". The factors are used as described on the US Sailing website (look under Offshore for a link to PHRF and Time on Time Scoring)

GUI Look and Feel:  In the dropdown box you can select the look and feel of Javascore.(Restart of Javascore recommended)

Locale: In the dropdown box you can select the language. The Javascore language defaults to the language of the operating system if it is Windows.(Restart of Javascore recommended)

Note: Effective with Javascore version 7.0, the old settings for identifying a browser for reporting have been removed as Java now handles this "natively" and user input is no longer needed.
