JavaScore - Split by Seeding

Use this dialog to split a fleet into new divisions during a qualifying series.

This "by seeding" split will assign boats to their new divisions based on the current overall position in the selected stage. It will place the 1st place boat in the first division listed, the 2nd to the second division listed and so on. At the last division the assignment reverses.

For example, if you want to perform the split at the end of day 1 into 2 new divisions called "Yellow Day 2", and "Blue Day 2", then split will occur as follows:

1st place -> Yellow Day 2
2nd place -> Blue Day 2
3rd place -> Blue Day 2
4th place -> Yellow Day 2
5th place -> Yellow Day 2
etc...

If you want to perform the split at the end of day 1 into 3 new divisions called "Yellow Day 2", and "Blue Day 2" and "Green Day 2"

1st place -> Yellow Day 2
2nd place -> Blue Day 2
3rd place -> Green Day 2
4th place -> Green Day 2
5th place -> Blue Day 2
6th place -> Yellow Day 2
7th place -> Yellow Day 2
8th place -> Blue Day 2
etc...

The order of the divisions is important and should be specified in the Sailing Instructions. Once the panel is completed and the user presses OK, the data will be validated, and if there are no errors, you will get a final confirmation check to make sure you want to go through with the split.

If you create a new split, and later decide you want to delete it. Go into "Divisions | Subdivisions" and delete the divisions. If you have scored races involving those divisions, you should also delete those races or their scores will be incorrect.

The panel has the following fields:

Stage to Split: This is the stage in which the split is to occur. In the stage, the "Is Qualifying Stage" must be selected. Only stage with the "Is Qualifying" option selection will be show in the drop down box.

Division to Split: This is the division that is to be split into qualifying subdivisions.

Number of new divisions: The number of new qualifying divisions to be created.

New Division Names: The names of each new division. These names must be unique within the regatta. For example, you cannot have two "Red" divisions, use something like "Red Day 1" for the first day's splits, then "Red Day 2" for the next Red split.
