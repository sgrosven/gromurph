Java Score - SubDivision Screen

This panel edits information about the subdivisions participating in a regatta.   

A subdivision is a subset of a Starting Divisions.  Members of a sub-division race with their parent starting division and a scored as part of both the starting division and as part of the subdivisions.   An example of a sub-division might be scoring Juniors within a larger Open Division.

See also the General Intro page for more information on using Fleets and SubDivisions.

This screen as two main areas.  On the left are the names of all subdivisions in this regatta.  On the right, the details of a selected subdivision.  Press Add to add a new subdivision.  Select one or more races, and press Delete to delete a subdivision.

Within a subdivision, you currently have the following fields to set:

SubDivision Name: name of the class.  Anything you like.  This will be the name used on all reports for this subdivision.

Parent Division:  Select from the drop-down list of all starting classes in the regatta, which class your subdivsion members will come from. 

Race score points add-on:  Use this to "add-on" points for each race any boat in this subdivision races in. This is used mostly commonly for events like Laser Masters where the "Apprentice" master sailors typically get 3 points added on, "Masters" get 2 points, "Grand Masters" get 1 point, and "Great Grand Masters get 0 points". NOTE that these add-ins are non-standard and must be well defined in the sailing instructions for the event!

Score entrants separately from parent class:  Check this box if you want the members of the sub-division to be re-scored separately from their finish within their parent class.  If for example, you have a sailor, Pat, in the Junior subdivision within the Open starting division.  In race 1,  Pat gets a 5th overall, but is the first Junior sailor to finish.  Pat will get a 5th within the Open class.  If  Pat's score in the Junior subdivision should also be a 5th, leave this box unchecked.  If Pat's should get a 1st in the Junior subdivision, then check this box.  (Strong suggestion: Ensure that what you select is backed up by the sailing instructions)

Can entrants belong to other subdivisions:  Check this box if entries can be entered in more that one subdivision. 
    Sometimes, subdivisions like Opti Blue, Green, and White, there should be no overlap (a sailor cannot be both a Blue and a Green Opti sailor at the same time).  In this case, make sure this box is unchecked.
    On the other hand, the subdivisions might be something like "Women","Senior","Junior" within the overall "Open" class.  A young female sailor would be eligible for both "Women" and "Junior" divisions.  In these cases, make sure this check box is selected.

Qualifying/Final/Medal/Scoring Group Type:  If you are splitting the entries into qualifying groups then into Final groups.  Click "Qualifying" if this subdivision is one of the qualifying sub-divisions, otherwise click final if its part of the final series sub-divisions. If you further split your final series into a medal race, select Medal (You normally only do this for Sailing World Cup events or Olympic-style racing)

If the group is used for sub-group not related to the qualifying final divisions, then select "Scoring" (eg the red/white/blue subidivison in the Opti class, or the "Corinthian" division in the Melges class
