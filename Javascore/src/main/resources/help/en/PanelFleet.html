Java Score - Fleets Screen

This panel edits information about the fleets participating in a regatta.   

A fleet is a combination of several Starting Divisions.  All members of each Starting Division in a Fleet are combined into the fleet for "overall" scoring.  Some examples of fleets include:

  • Several individual PHRF classes could be combined into an overall PHRF fleet. 
  • A large one-design regatta could be split into several "qualifying groups" which race in various combinations. 

See also the General Intro page for more information on using Fleets and SubDivisions.

This screen as two main areas.  On the left are the names of all fleets in this regatta.  On the right, the details of a selected fleet.  Press Add to add a new fleet.  Select one or more races, and press Delete to delete a division.

Within a fleet, you currently have the following fields to set:

Fleet Name: name of the class.  Anything you like.  This will be the name used on all reports for this fleet.

Starting Divisions:  There should be a check box for each starting division in the regatta.  Click on those divisions that should be part of this fleet. 

    Select All:  Check here to quickly select all starting divisions to be part of the fleet.

    Clear All:  Click here to quickly remove all starting divisions from the fleet.

Score together if same start time:  Check this box if you want starting divisions that start at the same time to be scored as a single starting class.  You would select this if you were scoring a qualifying regatta where say "green" and "pink" classes were racing one race together and the starters in those two divisions should be combined into a single race.

Handle races with different lengths:  Select how you want races involving starting divisions that sail a different length race handled.  Currently there are two options: drop that race from the fleet series score, or score them as is and ignore the difference.
