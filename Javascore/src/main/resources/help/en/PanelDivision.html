Java Score - Master Divisions Screen

This panel edits information about all known classes or divisions.  When you create a regatta, you can select among this list of classes to determine what classes are participating in the regatta. 

This screen as two main areas.  On the left are the names of all known classes.  On the right, the details of a selected class

Press Add to add a new division

Select one or more races, and press Delete to delete a division.

Note that editing this information will affect only the master list of classes, once a class has been entered into a regatta, its information is frozen for that regatta.

The details about a class are:

Class Name: name of the class.  Anything you like.  Although be aware that if you import entries via the File | Import menu, the name of the class in the import data must be an exact match on a class in this list.

Rating System: the rating system to be used with this class.  Currently, Javascore supports one design, PHRF, MORC (which is their newer seconds/mile basis), and the Chesapeake Bay's Multihull formula (which is a pretty straight forward Time-on-Time formula).

1D Class Name: (for one design classes only)  The name of the one design class.  Most of the time this will be the same as the Class Name.  However, you might have for example a large Laser regatta in which you have Laser Masters, Laser Youths, and Laser Open.  In that case, your class name in the upper field might be "Masters", "Youths", "Open", which the 1D Class Name for all classes would be "Laser".
    This field is a "combo box" if you see the class you want in the list, go ahead a select it, if not, just type in the name of the class you do want

Fastest Rating: (Handicap classes only) The fastest allowed rating in this division.

Slowest Rating: (Handicap classes only) The slowest allowed rating in this division.

When you press OK to close this dialog, the master division information is automatically saved.
