JavaScore - Split by Ranking

Use this dialog to split a fleet into new stages and divisions at the end of a stage. The split will be performed based on current overall stage ranking with the top boats going to the first division, the lower boats going into another division (or not advancing at all)

The splits must be specified in the Sailing Instructions.

Note that in this split, new Stages and new Divisions will both be created. Each stage will have one division in it, and the stage name and division names will be the same. The new stages' scoring options will be, by default, the same as their previous stage.

If you create a new split, and later decide you want to delete it. Go into "Divisions | Subdivisions" and delete the divisions. Go into "Regatta | Stages" and delete stages. If you have scored races involving those divisions, you should also delete those races or their scores will be incorrect.

The panel has the following fields:

Stage to Split: This is the stage in which the split is to occur. This will also become the "Previous stage" for the new stages.

Division to Split: This is the division that is to be split into new stages and divisions.

Number of new divisions: The number of new stages/divisions to be created.

New Stage/Division Names: The names of each new stage/division. If a division or stage already exists, you will get an alert, and have the option to keep the existing stage/division (with all entries replaced) or cancel out.

Top Pos: The ranking for the top boat to be placed into each division. If some boats will not advance into a new stage/division, then place the ranking of the first boat to not advance in the bottom-most row ("Not advancing")

Create Carryover Race: If you want the new stage to include a carryover race where each boat's score is their ranking from the preceding stage (relative to other boats in their new stage), then check this box. The carryover race will be created in each new stage, and the stage's scoring options will be set accordingly.
