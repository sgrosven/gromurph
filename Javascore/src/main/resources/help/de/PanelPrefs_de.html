Java Score - Einstellungen

Screenshot

Hier werden allgemeine Programmeinstellungen vorgenommen. Zu finden im Menü 'Datei'.

Versch.(iedenes) 1.Eintrag: Hiermit kann ein optionales Feld bei den Teilnehmerdaten definiert werden. Nach der Erstinstallation steht hier wohl "CBYRA#" oder "Rsa" oder etwas anderes. Das ist nur eine Standardeinstellung nach der Installation. Es empfiehlt sich hier "Club Crew ", aber es kann auch jeder andere Eintrag gewählt werden.

Versch.(iedenes) 2.Eintrag: Hiermit kann ein weiteres optionales Feld bei den Teilnehmerdaten definiert werden. Nach der Erstinstallation steht hier wohl "US Sailing#" oder "Mna" oder etwas anderes. Das ist nur eine Standardeinstellung nach der Installation. Es empfiehlt sich hier "DSV-Clubnr.", aber es kann auch jeder andere Eintrag gewählt werden.

Browser für Berichte Alle Berichte werden im HTML Dateiformat generiert und stehen für eine Anzeige im Browser sofort zur Verfügung. Natürlich können sie auch sofort zur Veröffentlichung im Web auf einen Webserver geladen werden.

System Standard:  In den meisten Fällen ist ein Standard Browser vom Betriebssystem vorgegeben. Zunächst sollte man es mit dieser Einstellung versuchen.

Interner Browser:  Javascore hat einen internen Browser, der geht immer.

Anderer: Wenn es ein anderer als einer der beiden erstgenannten Browser sein soll, so kann er hier gewählt werden.

grafische Oberfläche:  In der Auswahlbox kann man sich das 'Look and Feel' von Javascore einstellen.(Programm Neustart empfohlen)

Sprachversion: In der Auswahlbox kann man die Sprache umstellen. Javascore installiert sich zumindest bei Windows in der Standardsprache des Betriebssystems.(Programm Neustart empfohlen)
