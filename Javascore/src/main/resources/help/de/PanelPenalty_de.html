Hier werden Strafen oder auch eine Wiedergutmachung für Teilnehmer einer Wettfahrt bestimmt.

Es gibt drei Gruppen.

Nicht-im-Ziel Strafen:   Einem Zieldurchgänger kann nur eine dieser Strafen zugeordnet werden.  Wenn das der Fall ist, wird das Boot in den Wertungssystemen , die die Anzahl der Zieldurchgänger in die Wertung einbeziehen, nicht als Zieldurchgänger gelistet.

Disqualifikation Strafen:  Einem Zieldurchgänger kann nur eine dieser Strafen zugeordnet werden. Diese Strafen überschreiben komplett die Zieleinlauf Position eines Bootes. Wenn ein Boot diese Strafe zugeordnet bekommt, werden alle nachfolgenden Boote nach oben korrigiert.  

Andere Strafen und Wiedergutmachung:  Es ist unüblich, aber Boote können mehr als eine der nachfolgenden Strafen als Kombination zugeordnet bekommen. Einige Kombinationen ergeben aber keinen Sinn. Es bleibt hier Aufgabe der Auswertung eine sinnvolle Kombination zu finden.

  • Z-Flagge: Das Boot bekommt eine Strafe nach RRS 30.2.
  • Check-in Versäumnis: Das Boot bekommt eine Strafe weil es den vorgeschriebenen Check-in versäumt hat. ANMERKUNG: Dies ist nicht durch die RRS abgedeckt und das Regatta Komitee muss es in den Segelanweisungen vorsehen. Die Höhe der Strafe ist ein Prozentwert und wird im 'Wertungsoptionen Dialogfeld' festgelegt.
  • Punktedurchschnitt: Das Boot bekommt eine Wertung (z.B. auch Wiedergutmachung) entsprechend dem Durchschnitt anderer Zieldurchgänger gemäß RRS A10(a) (2001 Rules).  ANMERKUNG: Hier muss mit dem Protestkomitee abgestimmt werden, ob sie sich wirklich auf die A10(a) Definition beziehen. Wenn das Schiedsgericht anderes als A4(a) beabsichtigt, müssen die Punkte mit der Funktion 'Wiedergutmachung' per Hand eingegeben werden.
  • Prozentstrafe:  Das Boot bekommt eine 'Wertungsstrafe'. Gib rechts den entsprechenden Prozentwert (0-100) dazu im Feld 'v.H.' ein.
  • Zeitstrafe in Prozent: Das Boot bekommt eine Strafe für die Überschreitung einer Zeitvorgabe als Prozentwert. Gib rechts den entsprechenden Prozentwert (0-100) dazu im Feld 'v.H.' ein. Dieser wird vor Anwendung eines Handicaps eingerechnet.
  • Zeitstrafe als Zeitwert:  Das Boot bekommt eine Strafe für die Überschreitung einer Zeitvorgabe. Gib rechts den entsprechenden Zeitwert (hh:mm:ss) dazu im Feld 'Zeit' ein. Dieser wird vor Anwendung eines Handicaps eingerechnet.
  • Manuelle Punkteingabe: Das Boot bekommt Extrapunkte (z.B. bei Spaßregatten), die das Wertungsprogramm normalerweise nicht abhandelt. Der Punktbetrag muß dann im Feld 'Pkt' eingegeben werden. Ansonsten sollte diese Funktion sehr sparsam eingesetzt werden und es empfiehlt sich, die Anwendung unter 'Notiz' zu erläutern.
  • Wiedergutmachung: Das Boot hat vom Protest Komitee Wiedergutmachung zugestanden bekommen, anders als es gemäß A4(a) Durchschnittspunkte schon weiter oben beschrieben ist. Genau wie bei der 'Manuellen Punkteingabe' müssen die Punkte im 'Pkt' Feld definiert werden.
  • Anmerkung zu 'Manuelle Punkteingabe' und 'Wiedergutmachung': Wenn das Boot als ein Zieldurchgänger gelistet ist, sollte es eine richtige Zieleinlaufposition haben und Boote die dahinter ins Ziel kamen werden dann nicht unbedingt immer nach oben verschoben. Wenn dieses Boot nicht im Ziel war, sollte es eigentlich eine Nicht-im-Ziel Strafe wie MAN oder RDG bekommen.
  • Allgemeine Anmerkung: Mit den 2001 RRS haben sich viele Definitionen der Strafen geändert; wenn aber das alte 1997er Wertungssystem benutzt wird, werden die Strafen in Anlehnung daran gewertet, allerdings bleiben die Abkürzungen gemäß den 2001 Regeln.

Notiz: Hier werden alle zu dieser Strafe gehörenden Anmerkungen eingegeben. Sie werden als Fußnote bei den Regattaberichten ausgegeben. Man kann diese Anmerkungen dazu nutzen, um z.B. eine manuelle Punktwertung zu erklären, oder eine Schiedsgericht Entscheidung zu einem DSQ oder RDG.
