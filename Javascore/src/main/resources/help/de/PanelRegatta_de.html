Java Score - Regatta Dialogfeld

Screenshot

Hier werden Parameter für die Regatta als Ganzes eingegeben. Alle Felder sind optional, aber es wird dringend empfohlen einen passablen Namen für die Regatta zu vergeben.

Name: Der Name der Regatta. Danach wird ein Standard Name für die Speicherung der Regatta erzeugt.

Club: Der Name des austragenden Clubs.

Datum: Datum der Regatta.

Wettfahrtleiter: Der Name des Wettfahrtleiters.

Schiedsgericht: Die Namen der Schiedsgerichtsmitglieder.

Benutze Bug Nummern: Auswählen wenn Bugnummern für diese Regatta benutzt werden.

Ergebnisse sind abgeschlossen: Hier muss ein Haken gesetzt werden, wenn die Ergebnisse für eine Regatta abgeschlossen sind, alle Wettfahrten komplett und alle Proteste abgehandelt. Es wird sofort ein aktualisierter Satz von Berichtsseiten erstellt, der das Attribut "abgeschlossen" bekommt  und die Regattadatei wird gespeichert und ebenfalls mit dem Attribut "abgeschlossen versehen, welches verhindert das die Regatta unabsichtlich nochmal verändert werden könnte. Erst wenn der Haken bei "Ergebnisse sind abgeschlossen" wieder entfernt wird kann die Regattadatei wieder bearbeitet werden.

Wertungssystem: Auswahlfeld - Hier wird das Wertungssystem für alle Wettfahrten der Regatta eingestellt. Momentan unterstützt sind ISAF 2001-2004 Lowpoint, DN Iceboat Lowpoint, Lightning Lowpoint, Snipe Lowpoint und Lowpoint-AYCWedPenalties. 

Wertungsoptionen: Schaltfläche - Ruft ein Feld auf, das weitere Optionen zur Regattawertung ermöglicht (z.B. Streicher)

Klassen/Flotten in dieser Regatta: Dies ist eine Liste aller startenden Klassen/Flotten. Diese Liste findet sich auch unter dem Memüpunkt Klassen/Flotten -> Klassen/Flotten.  

Hinzufügen  Schaltfläche - Hiermit kann eine neue Klasse/Flotte der Regatta hinzugefügt werden. Es wird eine Auswahlbox mit allen noch nicht zugeordneten Klassen/Flotten aufgerufen. Wenn eine neue Klasse/Flotte erzeugt werden soll muss dies über denMemüpunkt Klassen/Flotten -> Klassen/Flotten geschehen.

Löschen Schaltfläche - Hiermit wird eine Klasse/Flotte aus der Regatta entfernt. Boote, die schon einer der gelöschten Klassen zugeordnet waren, werden einer Klasse <nn> zugeordnet.  Sie erscheinen auch im Zieleinlauf Dialogfeld.

Bearbeiten Schaltfläche - Ruft ein Feld auf zur Bearbeitung einer Klasse/Flotte der aktuellen Regatta. Falls die Änderung einen Fehler ins System bringen könnte, wird eine Warnung ausgegeben.
