Java Score - Klassen/Flotten Dialogfeld

Screenshot

In diesem Feld kann man alle Klassen/Flotten bearbeiten.Wenn eine Regatta erstellt wird, kann man aus der Liste der Klassen/Flotten bestimmen welche an der Regatta teilnehmen sollen.
Es gibt zwei Hauptbereiche.  Links die Namen der Klassen/Flotten, rechts die Details der gewählten Klasse/Flotte.
Mit 'Hinzufügen' kann eine neue Klasse/Flotte hinzugefügt werden.

Zum Löschen kann man eine oder mehrere Klassen/Flotten markieren und 'Löschen' drücken.

Alle Änderungen in diesem Feld betreffen nur die globalen Einstellungen. Sind Klassen/Flotten schon einer Regatta zugeordnet können die darin enthaltenen Einstellungen nicht mehr global geändert werden.

Die Einzelheiten zu einer Klasse/Flotte:

Klassen/Flotte Name: Name der Einheitsklasse oder ein beliebiger gut zutreffender Name.  Es ist sicherzustellen das bei Import von Teilnehmern über  'Datei | Import Menü', der Name der Klasse/Flotte der Import Daten mit den Namen in dieser Liste übereinstimmen muss.

Rating System: Das Ratingsystem, das mit dieser Klasse benutzt wird.  Momentan unterstützt Javascore Einheitsklassen, PHRF, MORC und die Chesapeake Bay's Multihull Formel.

Einheitsklassen: (Nur wenn auch Einheitsklassen gewählt sind)  In den meisten Fällen wird dies wohl der Name der Klasse/Flotte sein. Aber in anderen Fällen kann auch eine andere Bezeichnung gewählt werden, z.B. wenn in einer großen Laser Regatta nach Masters, Jugendlichen und Offener Klasse unterschieden werden soll. Für diesen Fall kann dann "Masters", "Jugend" und "Offen" als Einheitsklassenname eingetragen werden und der Einheitsklassenname für alle Klassen kann dann "Laser" sein.
Dieses Feld ist eine Combo-Box , d.h. es kann eine Klasse in der Drop-Down Box ausgewählt oder ein neuer Name eingetragen werden.

Fastest Rating: (nur bei Handicap Klassen) Der Fastest Rating-Wert dieser Klasse.

Slowest Rating: (nur bei Handicap Klassen) Der Slowest Rating-Wert dieser Klasse.

Alle Eingaben werden automatisch durch Klicken auf OK gespeichert.
