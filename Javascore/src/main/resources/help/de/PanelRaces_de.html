Java Score - Wettfahrten Dialogfeld

Screenshot

In diesem Feld werden Wettfahrten und deren Zieleinläufe bearbeitet.  Es gibt zwei Bereiche. Links werden alle Wettfahrten dieser Regatta aufgelistet und rechts stehen die Details der jeweils markierten Wettfahrt.   

Zum Eingeben der Zieleinläufe muß auch die jeweilige Wettfahrt markiert sein.

Mit 'Hinzufügen' kann eine Wettfahrt erstellt werden.

Mit 'Löschen' kann man die jeweils markierte(n) Wettfahrt(en) löschen.

Wettfahrtdetails:

Name: Name der Wettfahrt.  Standardmäßig erscheint bei Neuerstellung einer Wettfahrt eine Zahl (z.B. eine 5, wenn es die 5.Wettfahrt ist). Es muß nicht unbedingt eine Zahl sein, aber es ist dringend empfohlen die Zahl oder einen Wettfahrtnamen beginnend mit dieser Zahl zu wählen. Das trägt wesentlich zu eindeutigen Sortierungen in den Berichten bei.

Start Datum: Das Startdatum der Wettfahrt. Für neue Regatten wird das Standarddatum auf heute gesetzt. Zur Auswahl eines anderen Datums klappt normalerweise ein Kalender auf, in dem nur das Datum angeklickt werden muß. Falls das auf Grund einer älteren Javaversion nicht passiert, kann man das Datum auch per Hand eingeben (Bitte in diesem Beispielformat: 13-Jul-2005).

Für jede Klasse/Flotte in der Regatta sollte die nachfolgende Information eingegeben werden:

Einschliessen?:  Diese Box muß markiert sein, wenn diese Klasse/Flotte in dieser Wettfahrt segelt (Achtung: Als Standardeinstellung wird angenommen, das alle Klassen/Flotten dieser Regatta starten).

Startzeit  Startzeit für eine Klasse/Flotte. Diese Feld ist zwar optional, dient aber zur Definition von gemeinsam startenden Qualifikationsgruppen o.ä. und zur Kalkulation von Handicap korrigierten Zeiten.

Bahnlänge: Die Länge der Wettfahrtstrecke. Dieses Feld ist ebenfalls optional, kann aber zur richtigen Behandlung von Handicap-Booten gebraucht werden.

Zieleinlauf bearbeiten Schaltfläche:  Hiermit ruft man das Dialogfeld zur Eingabe und Bearbeitung von Zieleinläufen und Strafen auf.

Rundungen bearbeiten Schaltfläche:  Dies ist optional, eröffnet aber die Möglichkeit, die Reihenfolge der Boote beim Runden der Bahnmarken einzugeben. Die Bahnmarken sind numeriert und für jede können die Rundungen eingegeben werden. (Anmerkung: Hier eingegebene Strafen werden ignoriert)

Anmerkungen bearbeiten Schaltfläche: Dies ist eine optionale Möglichkeit Kommentare über die Wettfahrt einzugeben. Diese werden bei den jeweiligen Regattaberichten oben eingefügt. Es können HTML Tags zur Formattierung benutzt werden.

Bedingungen: Dieser Set von Auswahlbuttons wird für PHRF Time-on-Time Wertung gebraucht. Wähle die entsprechende Windbedingung. Diese Felder sind verborgen, wenn keine PHRF Klassen für PHRF Time on Time Wertung vorgesehen sind.
