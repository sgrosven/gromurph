Hiermit werden Gruppen für eine Regatta bearbeitet und erstellt

Eine Gruppe ist eine Kombination von mehreren Klassen/Flotten . Alle Teilnehmer jeder Klasse/Flotte in einer Gruppe sind zusammengefasst für eine Gruppen "Gesamt"-Wertung. Die Wertung innerhalb einer Klasse/Flotte erfolgt ebenfalls. Beispiele:

  • Einige individuelle Klassen können zu einer Gesamt-Klasse zusammengefasst werden. 
  • Eine große Einheitsklassen Regatta kann aufgeteilt werden in mehrere "Qualifikationsgruppen" die in unterschiedlichen Kombinationen segeln können.

Siehe auch die Kurzeinführung zur Regattaauswertung für weitere Informationen zum Einsatz von Gruppen und Teilgruppen.

Dieses Feld hat zwei Hauptbereiche.  Links stehen die Namen aller Gruppen dieser Regatta. Rechts stehen die Details der gewählten Gruppe. Mit 'Hinzufügen' kann eine neue Gruppe erzeugt werden und mit 'Löschen' kann man die Gruppe(n) entfernen, die vorher markiert wurde(n).

In einer Gruppe sind folgende Felder zu definieren:

Gruppen Name: Name der Klasse.  Frei wählbar, dieser Name wird in allen Berichten benutzt.

Klassen/Flotten:  Für jede Klasse/Flotte dieser Regatta ist eine Checkbox da. Diejenigen, die zur Gruppe gehören sollen, müssen markiert sein. 

    Alle auswählen :  Hiermit werden alle Klassen/Flotten einer Regatta in diese Gruppe übernommen.

    Alles zurücksetzen:  Hiermit werden alle Klassen/Flotten der Gruppe schnell zurückgesetzt.

Zusammen werten bei gleicher Startzeit:  Diese Checkbox muß gewählt werden, wenn Klassen/Flotten, die zur gleichen Zeit starten, als eine Startgruppe gewertet werden sollen. Z.B. bei Qualifikations-Wettfahrten oder aufgeteilten größeren Feldern.

Wettfahrten verschiedener Streckenlänge:  Hier wird bestimmt, wie Wettfahrten abgehandelt werden, die Klassen/Flotten beinhalten die unterschiedliche Bahnlängen segeln:
- nicht in Gruppenwertung einschliessen, oder
- Wertung wie Zieleinlauf ohne Berücksichtigung der verschiedenen Bahnlängen.
