Java Score - Wertungsoptionen

Screenshot

Hier können optionale Wertungseinstellungen für eine Low Point Regatta vorgenommen werden.

Ziel Zeitlimit Strafe: Klicke auf die Checkbox die die Wertung für Boote bezeichnet, die  innerhalb eines vordefinierten Zeitlimits zum ersten Boot ins Ziel kommen (und später mit einer TLM Strafe gewertet werden). ANMERKUNG: Dieses muss in den Segelanweisungen bekanntgegeben werden. Wenn es dazu keine Aussage gibt, auf dem voreingestellten Wert DNF belassen.

Streicher: Es gibt drei Muster für die Einstellung der Anzahl von Streichern.

  • Bei Mindestzahl aller Wettfahrten: Dies ist die gebräuchlichste Art Streicher zu definieren. Für 1,2 oder 3 Streicher wird die Mindestzahl von gesegelten Wettfahrten eingestellt. Voreinstellung  ist 1 Streicher nach 2 Wettfahrten.
  • Ein Streicher für je <x> abgeschlossene Wettfahrten:  In einigen längeren Serien können die Segelanweisungen beispielsweise vorgeben das für je 3 gesegelte Wettfahrten ein Streicher gegeben wird. in diesem fall muss hier die Checkbox gewählt werden und die Anzahl der Wettfahrten pro einem Streicher eingetragen werden.
  • Nur die besten <x> Wettfahrten zählen: Eine andere gebräuchliche Art ist, dass nur die besten, beispielsweise 4, Wettfahrten zählen. Wenn es dann z.B. 8 Wettfahrten gibt, entspricht das 4 Streichern.
  • ANMERKUNG: Regatten ohne Streicher:  Wähle "Bei Mindestzahl aller Wettfahrten" und gib 0 für 1 Streicher ein.

Check-in Versäumnis Strafe: Manchmal schreiben Segelanweisungen den Booten ein verbindliches Checkin vor und schreiben eine Strafe vor. Gib den Prozentsatz für die Strafe ein, als Zahl 1-100. Wenn dann die Wettfahrtergebnisse der Boote eingegeben werden, muss den Booten die eine Checkin Strafe bekommen sollen, diese zugeordnet werden.

Wertung als "Serie länger als eine Regatta":  Wähle "Ja", um Strafen entsprechend RRS A9 behandeln zu können. Dies bezieht sich nur auf Wertungssysteme die auf den ISAF 2001-2004 Regulations basieren.
